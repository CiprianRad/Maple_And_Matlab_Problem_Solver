# Maple & MATLAB Problem Solver App

## 📌 Overview
This repository contains a TypeScript-based web application designed to help mathematics students learn how to use essential mathematical software tools. The app solves simple mathematical problems and showcases the solutions side-by-side in both **MATLAB** and **Maple**. Additionally, it provides a well-formatted **LaTeX** documentation of the problems, serving as a practical example of how to write high-quality mathematical reports.

## 📂 Repository Structure
The project is structured as a modern web application:

* **`components/`** - Reusable UI components for the web interface.
* **`services/`** - Core logic handling data fetching, solver integrations, or state management.
* **`App.tsx`** & **`index.tsx`** - The entry points and main application layouts.
* **`package.json`** & **`vite.config.ts`** - Configuration files for dependencies and the Vite build tool.
* **`types.ts`** - TypeScript type definitions used throughout the app.

## 🛠️ Features Demonstrated
Side-by-side code solutions for problems in Maple and MATLAB.

Integrated LaTeX outputs to teach students mathematical formatting and documentation.

Built with modern web technologies: React, TypeScript, and Vite.

Interactive web UI to easily navigate between problem sets and code views.
